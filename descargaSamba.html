Descarga Samba

El código fuente de Samba se distribuye a través de https. Ver el área de descarga a través de HTTPS . El archivo que probablemente desee se llama samba-lastest.tar.gz . Las versiones antiguas están disponibles en los archivos de Samba.

La clave pública GPG de distribución de Samba se puede utilizar para verificar que las versiones actuales no hayan sido manipuladas. Usando GnuPG, simplemente descargue la distribución fuente de Samba, la firma tarball y la clave pública de la distribución Samba. Entonces corre

        $ gpg --import samba-pubkey.asc
        $ gunzip samba-version.tar.gz
        $ gpg --verify samba-release.tar.asc
        gpg: Signature made Tue 20 Nov 2007
        07:12:04 PM CST using \
        DSA key ID 6568B7EA
        gpg: Good signature from "Samba Distribution Verification Key \
        ‹samba-bugs@samba.org›

Se recomienda que también revise la lista de parches para las versiones actuales. Para obtener información sobre las versiones de seguridad de Samba, consulte nuestra página de seguridad .

binarios

Todas las principales distribuciones de Linux y Unix libres tienen Samba como paquete nativo. Consulte el paquete o sistema de puertos de su distribuidor para obtener una instalación nativa de samba en su sistema.

https://samba.plus/ ofrece paquetes Samba para SLES, RHEL, Debian y AIX.

http://en.openSUSE.org/Samba ofrece paquetes Samba para todos los productos SuSE Linux (incluido SLES).

Fuentes de Git

También puede recuperar las fuentes utilizando el sistema de control de código fuente GIT . La ventaja de buscar a través de GIT es que puedes actualizar tus fuentes en cualquier momento usando un solo comando. Consulte las instrucciones de Git .

Herramientas

  • Administradores de GUI de Samba
  • Wireshark (decodifica NetBIOS, SMB/CIFS y MS-RPC)
  • tcpdump(rastreador de paquetes de línea de comando)
  • Herramienta de auditoría NetBIOS (NAT)
  • wsdd es un demonio de descubrimiento de servicios web (WSD) escrito en Python
  • wsdd2 es un demonio de descubrimiento WSD/LLMNR escrito en C
  • nbfw, el reenviador NetBIOS (obsoleto)
